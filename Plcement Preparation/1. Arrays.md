# Arrays in Java for Placement Preparation

## 1. What is an Array?

An **array** is a data structure that stores a fixed number of elements of the **same data type** in **contiguous memory locations**.

* Stores multiple values using a single variable name.
* Elements are accessed using an **index**.
* Array indexing starts from **0**.

### Example

```java
int[] numbers = {10, 20, 30, 40, 50};
```

| Index | Value |
| ----- | ----- |
| 0     | 10    |
| 1     | 20    |
| 2     | 30    |
| 3     | 40    |
| 4     | 50    |

---

## 2. Definition of Array

An array is a **linear data structure** where elements are stored sequentially and can be accessed directly using their index.

### Mathematical Representation

```
ArrayName[Index]
```

Example:

```
A[0], A[1], A[2], ..., A[n-1]
```

---

## 3. Characteristics/Properties of Arrays

### 1. Fixed Size

Once an array is created, its size cannot be changed.

```java
int[] arr = new int[5];
```

---

### 2. Homogeneous Elements

All elements must have the same data type.

```java
int[] arr = {1,2,3};     // Valid
// int[] arr = {1,"abc"}; // Invalid
```

---

### 3. Contiguous Memory Allocation

Elements are stored in adjacent memory locations.

```
1000 -> arr[0]
1004 -> arr[1]
1008 -> arr[2]
```

---

### 4. Random Access

Any element can be accessed directly using its index.

```java
arr[3];
```

Time Complexity:

```
O(1)
```

---

### 5. Index Based Structure

For an array of size n:

```
First index = 0
Last index = n-1
```

---

### 6. Static Data Structure

The size is decided during creation and remains fixed.

## 8. Operations on Arrays

### 1. Traversal

```java
int[] arr = {10,20,30,40,50};

for(int i=0;i<arr.length;i++)
{
    System.out.println(arr[i]);
}
```

Time Complexity:

```
O(n)
```

---

### 2. Insertion

```java
int[] arr = new int[6];

arr[0]=10;
arr[1]=20;
arr[2]=30;
arr[3]=40;

for(int i=4;i>2;i--)
{
    arr[i]=arr[i-1];
}

arr[2]=25;
```

Before:

```
10 20 30 40
```

After:

```
10 20 25 30 40
```

Time Complexity:

```
O(n)
```

---

### 3. Deletion

```java
int[] arr={10,20,30,40,50};

int pos=2;

for(int i=pos;i<arr.length-1;i++)
{
    arr[i]=arr[i+1];
}
```

Output:

```
10 20 40 50
```

Time Complexity:

```
O(n)
```

---

### 4. Searching

#### Linear Search

```java
public class Main {
    public static void main(String[] args) {

        int[] arr={10,20,30,40,50};
        int key=30;

        for(int i=0;i<arr.length;i++)
        {
            if(arr[i]==key)
            {
                System.out.println("Found at "+i);
                break;
            }
        }
    }
}
```

Complexity:

```
Best: O(1)
Worst: O(n)
```

---

#### Binary Search

```java
public class Main {
    public static void main(String[] args) {

        int[] arr={10,20,30,40,50,60};

        int low=0;
        int high=arr.length-1;
        int key=40;

        while(low<=high)
        {
            int mid=(low+high)/2;

            if(arr[mid]==key)
            {
                System.out.println("Found");
                break;
            }
            else if(arr[mid]<key)
            {
                low=mid+1;
            }
            else
            {
                high=mid-1;
            }
        }
    }
}
```

Complexity:

```
O(log n)
```

## 10. Disadvantages of Arrays

### 1. Fixed Size

```java
int[] arr = new int[100];
```

Cannot increase size later.

---

### 2. Insertion is Expensive

Need to shift elements.

```
O(n)
```

---

### 3. Deletion is Expensive

Need to shift elements.

```
O(n)
```

---

### 4. Memory Wastage

```java
int[] arr = new int[1000];
```

If using only 10 elements, remaining memory is wasted.

---

### 5. Homogeneous Data Only

Cannot store multiple data types.

## 12. Important Array Concepts for Placements

### Two Pointer Technique

```java
int[] arr={1,2,3,4,5};

int left=0;
int right=arr.length-1;

while(left<right)
{
    int temp=arr[left];
    arr[left]=arr[right];
    arr[right]=temp;

    left++;
    right--;
}
```

---

### Prefix Sum

```java
int[] arr={1,2,3,4,5};
int[] prefix=new int[arr.length];

prefix[0]=arr[0];

for(int i=1;i<arr.length;i++)
{
    prefix[i]=prefix[i-1]+arr[i];
}
```

Output:

```
1 3 6 10 15
```

---

### Sliding Window

```java
int[] arr={1,4,2,10,2,3,1,0,20};
int k=4;

int sum=0;

for(int i=0;i<k;i++)
    sum+=arr[i];

int max=sum;

for(int i=k;i<arr.length;i++)
{
    sum=sum-arr[i-k]+arr[i];
    max=Math.max(max,sum);
}

System.out.println(max);
```

## 14. When Should We Use Arrays?

Use arrays when:

* Size of data is known beforehand.
* Fast access is required.
* Data is homogeneous.
* Memory efficiency is important.
* Frequent insertions/deletions are not required.
